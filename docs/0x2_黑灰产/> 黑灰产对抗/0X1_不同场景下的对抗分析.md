# 场景下的风险分析

此篇主要对**不同业务场景**下的风险分析方法进行整理。



## 1. “垃圾注册”风险识别[1]

### 1.1 手法分析

黑产注册小号常见的套路是[1]：

-   使用接码平台提供的**虚拟号码**
-   使用模拟器、群控设备、云手机等模拟**设备环境**

-   通过**脚本调用**进行批量注册



 ### 1.2 思路

#### 从数据名单出发

-   手机号画像：主要依赖于黑名单库的形式
-   IP画像：识别风控模型中识别为**垃圾注册IP、代理IP与机房IP**的相关IP

#### 从设备端出发

-   识别**设备指纹**的风险
    -   模拟器
    -   安装了的作弊工具设备
    -   参数异常设备
    -   参数篡改设备
    -   疑似刷机设备

#### 从行为上出发

-   识别异常行为
    -   通过对平台的历史指标计算得到**频繁注册的设备、IP或手机号**等
